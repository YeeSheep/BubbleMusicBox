# BubbleMusicBox
# 介紹
將光感測器連接到樹梅派，然後把裝置放在盒子裡，利用盒子的開闔判斷有沒有光源。如果光大於200，LED會亮，泡泡機會啟動吹泡泡，音樂也會播出來。當沒有光的時候，LED不亮，泡泡機停止，音樂也會停止。
# 設備
- Raspberry Pi 3
- 光照度感測模組
- 伺服馬達
- 蜂鳴器
- LED 
- 風扇
- 麵包版
